# DocForge-Learning-Tasks

This repository contains the tasks for the learning phase of the project.

## 📌 How this repo works

* Each task will have its **own folder**.
* Inside every task folder, you will find a **README.md** containing:

  * task instructions
  * resources (if any)
  * expected outcome

## ✅ What you need to do

1. Complete each task in order.
2. After finishing a task, create a file named **explanation.md** inside that task’s folder.
3. In `explanation.md`, briefly write:

   * what you understood from the task
   * key concepts learned
   * any challenges faced (optional)

This helps us track your progress and ensures you understand the concepts, not just complete the code.

## 💡 Notes

* Make sure to push your work regularly to GitHub.
* Keep explanations short, clear, and in your own words.
* If you have doubts, feel free to ask in the group or DM a mentor.

Happy learning 🚀
